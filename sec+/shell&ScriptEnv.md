# Shell and Scripts Environment

## SSH

1. Encrypted method to remote connect to a machine
1. Port 22; telnet is 21 but not encrypted

## Powershell

1. Commandline for system admins
1. .ps1 is the file extension
1. Good for automation and admin stuff + active directory stuff

## Python

1. .py file extension
1. Good for automation + cloud
1. Good scripting tool

## OpenSSL

1. Library + utils for SSL/TLS certs
1. Used to build x.509 certs
1. Manage certificate signing requests + revocation lists
1. Hash functions to encrypt + decrypt for other services
